# 📖 Quran Web App

A modern, responsive web application for reading and searching the Quran with customizable display settings.

## 🌟 Features

- **Full Quran Text** - Complete Quran with translations
- **Search Functionality** - Search by Arabic text, translations, or Surah
- **Customizable Display** - Adjust font types and sizes for Arabic and translations
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Real-time Updates** - Settings sync across all pages instantly
- **Beautiful UI** - Modern emerald & teal color scheme with smooth animations
- **Persistent Settings** - Your preferences are saved locally
- **SSR Optimized** - Server-side rendering with Next.js 15+

## 🎨 Design

- **Color Scheme**: Emerald & Teal gradient theme
- **Typography**: Multiple Arabic fonts (Amiri, Noto Naskh Arabic, Cairo)
- **Animations**: Smooth transitions and entrance animations
- **Accessibility**: Proper semantic HTML and ARIA labels

## 🛠️ Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Font Management**: Next.js Font Optimization
- **Storage**: JSON

## 📦 Installation

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation Guide

```bash
# Clone the repository
git clone <repository-url>
cd quran-web

# Install dependencies
npm install

# Run development server
npm run dev

# Open browser
# Navigate to http://localhost:3000
```
